In this original design, every country has a line, and the flag is positioned based on the country’s rank in an ordered list of the relevant data. When ‘Life Expectancy Data’ is selected, for example, this ranking is done using the life expectancy column in the original data set. The leftmost country flag is the country with the highest value in the given data column. The green ‘plus’ sign and red ‘minus’ sign is meant to indicate this to the viewer.
